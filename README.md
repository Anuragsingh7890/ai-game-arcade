# Neural Arcade

Neural Arcade is an AI-themed mini game hub built with HTML, CSS, and JavaScript. It includes multiple browser games in one project with a shared dashboard, game menu, and session score tracking.

## Games Included

- Circuit Duel - Tic-tac-toe against a minimax-style AI opponent
- Data Snake - Classic snake with a data packet theme
- Memory Matrix - Card matching puzzle
- Signal Recall - Simon-style sequence memory game
- Latency Test - Reaction speed challenge
- AI Lab Quiz - Quick AI concept quiz

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage for session score persistence

## How To Run

Open `index.html` directly in your browser.

Or run a local server:

```bash
python -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173
```
